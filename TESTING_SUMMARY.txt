PASO 3: Configurar Google OAuth
────────────────────────────────
□ Pre-requisitos (solo una vez):
  
  1. Ir a: https://console.cloud.google.com
  2. Crear proyecto: "SIGO-Test"
  3. Habilitar: Google Calendar API v3
  4. Crear OAuth 2.0 Credentials (Web app):
     - Authorized redirect URIs:
       http://localhost:8000/calendario/callback
  5. Descargar credenciales JSON
  6. Configurar .env:
     
     GOOGLE_CLIENT_ID=<tu-client-id>
     GOOGLE_CLIENT_SECRET=<tu-client-secret>
     GOOGLE_REDIRECT_URI=http://localhost:8000/calendario/callback

  Si ya está configurado: ✅ Continuar
