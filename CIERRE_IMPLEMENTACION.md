## 🎬 CÓMO EMPEZAR AHORA MISMO

### Paso 1: Acceder
```
URL: http://localhost/SIGO/solicitudes/proceso
Requiere: Login + Rol (Directivo 2 o Admin 3)
```

### Paso 2: Explorar
```
1. Ves la bandeja principal
2. Puedes filtrar solicitudes
3. Clickea una para ver detalles
4. Revisa documentos, presupuesto, historial
5. Si todo OK → Firma digital
```

### Paso 3: Firmar
```
1. Clickea "Ver Resumen Completo"
2. Revisa el modal con detalles
3. Ingresa contraseña
4. Clickea "Firmar y Generar CUV"
5. ¡Listo! Solicitud aprobada ✅
```

## 💡 TIPS PARA USAR

```
PARA ACCEDER RÁPIDO:
  1. Ctrl+L en navegador
  2. Pega: localhost/SIGO/solicitudes/proceso
  3. Enter

PARA VER DOCUMENTOS:
  1. Clickea "Ver" en documento
  2. Se abre en nueva pestaña
  3. Puedes descargar si quieres

PARA FIRMAR:
  1. Revisa presupuesto (lado derecho, panel verde)
  2. Si dice "OK" → puedes firmar
  3. Clickea "Ver Resumen"
  4. Lee bien (¡ES IRREVOCABLE!)
  5. Ingresa contraseña
  6. Confirma

SI ALGO FALLA:
  1. Refresca página (F5)
  2. Borra cache (Ctrl+Shift+Supr)
  3. Prueba otro navegador
  4. Contacta soporte si persiste
```
