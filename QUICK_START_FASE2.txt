## ❌ Si falla algo

### SQL no ejecuta:
- Verifica que estés en base `BD_SIGO`
- Cierra otras pestañas de SSMS
- Reconecta

### No veo el folio:
```sql
SELECT * FROM Solicitudes WHERE folio = 1000
-- Debe retornar 1 registro
```

### Error de permisos:
```sql
SELECT fk_rol FROM Personal 
WHERE fk_id_usuario = (SELECT id_usuario FROM Usuarios WHERE nombre = 'dora1')
-- Debe retornar: 2
```

### Botón no funciona:
```bash
php artisan view:clear
php artisan cache:clear
# Recarga página (Ctrl + Shift + R)
```
