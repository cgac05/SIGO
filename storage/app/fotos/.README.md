# Directorio de Fotos de Usuarios

Este directorio almacena las fotos de perfil para:
- **Beneficiarios** - Fotos de cédula o DNI
- **Personal administrativo** - Fotos de identificación

## Estructura

```
storage/app/fotos/
├── beneficiarios/
│   └── {curp}.jpg
├── personal/
│   └── {numero_empleado}.jpg
└── .gitkeep
```

## Cómo acceder a las fotos

Las fotos están disponibles públicamente en:
```
/storage/fotos/beneficiarios/{curp}.jpg
/storage/fotos/personal/{numero_empleado}.jpg
```

## Métodos en el modelo

```php
// Obtener URL de la foto del usuario
$usuario->getFotoUrl(); 

// Retorna:
// 1. Foto local (storage/fotos/) si existe
// 2. Google Avatar si existe
// 3. Avatar por defecto (images/avatar-default.png)
```
